A task that represents the asynchronous operation. Upon completion, the task's result is a Document instance that represents the loaded document.
Remarks
A loaded document must have an absolute URI. Once a load operation has completed successfully, subsequent requests that resolve to the same URI are expected to return the same Document reference, although loaders are not required to manage document caches of unlimited size.
