ClearScript Library Reference

The ClearScript library makes it easy to add scripting to your .NET applications.
Namespaces
NamespaceDescription
Microsoft.ClearScriptThe Microsoft.ClearScript namespace contains types that support all script engines.
Microsoft.ClearScript.JavaScriptThe Microsoft.ClearScript.JavaScript namespace contains types that provide access to JavaScript-specific features.
Microsoft.ClearScript.V8The Microsoft.ClearScript.V8 namespace contains types that provide access to the V8 high-performance open-source JavaScript engine.
Microsoft.ClearScript.V8.FastProxyThe Microsoft.ClearScript.V8.FastProxy namespace contains types that support low-overhead access to host resources from the V8 JavaScript engine.
Microsoft.ClearScript.WindowsThe Microsoft.ClearScript.Windows namespace contains types that provide access to Windows Script engines such as JScript and VBScript in a desktop environment.
Microsoft.ClearScript.Windows.CoreThe Microsoft.ClearScript.Windows.Core namespace contains types that provide access to Windows Script engines such as JScript and VBScript.
