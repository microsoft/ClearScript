ClearScript Library Reference

The ClearScript library makes it easy to add scripting to your .NET applications.
Namespaces
NamespaceDescription
Microsoft.ClearScript
The Microsoft.ClearScript namespace contains types that support all script engines.
Microsoft.ClearScript.JavaScript
The Microsoft.ClearScript.JavaScript namespace contains types that provide access to JavaScript-specific features.
Microsoft.ClearScript.V8
The Microsoft.ClearScript.V8 namespace contains types that provide access to the V8 high-performance open-source JavaScript engine.
Microsoft.ClearScript.Windows
The Microsoft.ClearScript.Windows namespace contains types that provide access to Windows Script engines such as JScript and VBScript.
