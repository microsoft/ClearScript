Remarks
For maximum compatibility with hosts that predate an inadvertent breaking change in ClearScript 5.4.1, values greater than 1048576 (1 TiB) are assumed to be in bytes rather than MiB. For example, the values 16 and 16777216 both specify a limit of 16 MiB.
